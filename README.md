# qa-e2e-automation
This beginner-friendly project demonstrates end-to-end test automation covering both UI (Selenium) and API (Postman) tests. It includes a simple CI pipeline using GitHub Actions to run tests automatically on code push — simulating a real-world QA workflow.

🚀 Beginner-friendly End-to-End QA Automation Project

This project demonstrates UI and API test automation using:
- ✅ Selenium WebDriver (Java)
- ✅ TestNG (Test framework)
- ✅ Postman (API testing)
- ✅ GitHub Actions (CI pipeline)

---

## 📌 Features

- UI automation of login flow using Selenium
- API tests using Postman and optional Rest Assured
- Page Object Model structure
- CI/CD with GitHub Actions
- GitHub issues, PRs, and commit history to show real workflow
